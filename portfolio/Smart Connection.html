Smart Connection

Voor de leerlijn smart connection heb ik een aantal lessen/workshops gevolgd die aangeboden werden door de minor Smart Industry. Tijdens de eerste workshops heb ik de basis geleerd omtrent html, css, javascript en python. Deze opgedane kennis heb ik toegepast bij het opbouwen van dit portfolio. Zo heb ik een template gedownload en deze aangepast. Met behulp van tutorials op w3schools heb ik hier allerlei aanpassingen aan gedaan. Eigenlijk alles wat u ziet op dit portfolio is tot stand gekomen door de gevolgde workshops.

API's

Daarnaast heb ik geleerd hoe api's werken op websites en wat je hiermee kan. Een voorbeeld hiervan is deze pagina waarop ik met behulp van de api van open weather app het huidige weer in Nijmegen kan weergeven. Ook heb ik geprobeerd een widget hiervan op deze pagina hieronder te laten zien. Een andere api heb ik gebruikt om een beeld te krijgen van de huidige stand van zaken rondom het corona virus in Nederland. Deze kunt u hier zien.


Toepassingen buiten de minor

Deze basiskennis omtrent programmeren en het bouwen van een website kent natuurlijk oneindig veel mogelijkheden buiten deze minor om. Praktisch elk bedrijf heeft tegenwoordig wel een website. Hoe groot of klein je ook bent je kan eigenlijk niet meer zonder. Mijn aandacht gaat in dit geval ook niet uit naar deze basiskennis maar naar het gebruik van api’s in een bedrijfsomgeving. Bijvoorbeeld de corona api, deze kan door onder andere reisbureaus gebruikt worden om de huidige situatie in een land aan te geven aan hun klanten. Zo kan dat bijdragen aan een goede customer experience die de klanten hebben bij een reisbureau. Hetzelfde geldt natuurlijk voor de weer api. Omdat dit niet hele moeilijke materie is kan iedereen dit simpel op zijn of haar website plaatsen.

Daarnaast kan de weer api ook handig zijn voor intern gebruik bij bepaalde bedrijven. Denk hierbij aan bedrijven waarvan de medewerkers op verschillende plaatsen door het land, buiten onderhoud moeten plegen. Door deze api te gebruiken kan men bijvoorbeeld bij de eerstvolgende werkorder de weersvoorspelling van de betreffende locatie laten zien. Zo weten de medewerkers waar zij aan toe zijn en of zij bepaalde voorzorgsmaatregelen moeten nemen. Warm aankleden, extra zekering tegen harde wind, regenkleding of andere dingen. Mogelijk is dit zelfs te combineren met het systeem dat werkorders inplant. Wanneer de api waarden over het weer aangeeft die bepaalde grenswaarden overschrijdt kan het systeem automatisch de werkorder verplaatsen naar een andere datum die wel onder deze grenswaarden zit.
